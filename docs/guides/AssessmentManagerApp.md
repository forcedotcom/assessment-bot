# Open the Assessment Manager app

- Click on the App Launcher button and select the `Assessment Manager` option. (if you don't see it, click the View all link at the bottom)

![App](/docs/images/assessment-manager-app/app-selection.png?raw=true)

![App](/docs/images/assessment-manager-app/assessment-manager-app.png?raw=true)

- You will be redirect to the Assessment Manager app where you should be able to see the main tabs for the app
